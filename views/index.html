APIs and Microservices 5: File Metadata Microservice

User Stories:

  1. I can submit a FormData object that includes a file upload. Note: The form file input field has the name attribute set to "upfile". We rely on this for testing.
  2. When I submit a file I will receive the file name as name, MIME type as type, and size in bytes as size within the JSON response. Example: { "name": "test-file", "type": "application/octet-stream", "size": 75 }
